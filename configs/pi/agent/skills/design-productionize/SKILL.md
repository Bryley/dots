# Design Productionize

Stub skill for production handoff and implementation.

Expected responsibilities:
- Freeze the approved prototype direction before conversion.
- Break UI into reusable components with clear props/state boundaries.
- Port to the target framework/language (React, Svelte, Solid, etc.).
- Preserve design-system fidelity while improving code quality.
- Produce maintainable structure, naming, and implementation notes.
