# Tiny Task Runner

Tiny Task Runner is a small command line tool for running repeatable project tasks.

## Why

Teams often keep helpful commands in notes, chat threads, or shell history. Tiny Task Runner keeps those commands in one project file so everyone can find and run them. It also makes onbording easier by showing common workflows in a single place.

This project is useful for developers who want predictable local commands without learning a large automation framework.

## Usage

Run a named task with:

```sh
ttr run test
```
